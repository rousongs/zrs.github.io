本地编辑器vs Code

使用vs code写博客
2025-04-15    419字    1分钟    自然数    Blog

博客搭建好后，就可以用记事本打开content/posts下的md文件编辑文章内容了，但记事本毕竟太简陋了，也没预览功能，对于长期使用确实不是一个好的文章编辑工具。

那么现在有请我们强大的编辑软件vs code
vs code非常强大，强大到可以作为大多数编程软件的代码前端编辑工作。

如果vs code只用于hugo博客写作的话，配置起来非常的简单。先下载安装vs code，然后我们解决几个问题。
1.配置软件界面中文：打开VSCode，按住键盘上的“Ctrl+Shift+P”组合键，打开命令面板；在命令面板中输入“Configure Display Language”，然后点击“Co；nfigure Display Language”选项；在弹出的语言选择列表中，选择“zh-cn”，表示简体中文；然后重启VSCode，就可以变成了中文。
2.安装·Markdown All in One·插件，这样vs code就支持适时预览markdown页面
