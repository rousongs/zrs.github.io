Hugo的常用命令

Hugo的常用命令你都知道吗？
2025-04-08    74字    1分钟    自然数    Hugo

hugo
生成网站

hugo server
启动本地调试服务器

hugo server -D
启动本地调试服务器（包含Front Matter 中draft: true 的文章）

hugo new posts/blog/XXX.md
在posts/blog/目录下新建XXX文章
