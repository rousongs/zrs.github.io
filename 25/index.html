静态博客让人更专注于写作

相比动态博客，静态博客更让人专注于写作！
2025-04-27    478字    1分钟    自然数    Life

自从这个hugo站点架设好了之后，每天都琢磨着写点文字来充实博客的内容，虽然博主文学功底薄弱，但也是可能积少成多的。

之前折腾习惯了LNMP和LAMP，每当安装好wordpress或typecho之后，分心的内容太多，每天不是琢磨着敲打敲打LNMP的零部件就是琢磨琢磨博客的花哨的主题，反正心思就是不在写博客上。

突然有一天遇到了hexo/hugo这样的博客平台，老实讲这种模式对我来说有些新颖，也是有些门槛的，于是我把整个搭建过程都写成了博文。在苦心钻营很久搭建好之后，反而有一些要安心创作的踏实感。可能是暂时没有能力去调试或者自己二次开发主题吧，所以也没有动机去调试主题了。服务器使用github，也不需要维护和调试。这样我只需要安安心心地把博客的内容丰富起来，然后期待着每天都有 git add 和 git push 带来一丝丝的成就感。也许这才是博客的正统之道？

另外markdown语法还需要再学一学，即使是使用vs code写文章，也没有像typecho那样的简易编辑器，全靠自己手打标记，有些吃力。目前也就不讲究排版了，诠当文本文件来写了。

最后，祝自己多多创作，也祝大家多多创作。
